Data flow — one governed decision

sensors→ SAPIENT normalize→ IMM track→ deterministic gate <1ms→ human gate (Okta 2‑person)→ effector (SAPIENT)→ signed ledger → S3 WORM ⇢ async ⇢ Bedrock / Claude explains
The kill chain is deterministic and offline‑capable. When the link and cloud are denied (DDIL), steps 1‑7 keep running on the edge; only the Claude assessment (step 8) is deferred — it narrates, it never gates a shot. Datadog / Rapid7 / Okta are cross‑cutting; GitLab / Red Hat / Spectro Cloud / IBM build, run, and scale the platform.
